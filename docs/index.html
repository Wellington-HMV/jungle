Jungle

Mantenha o Windows ativo e o status do Teams/Slack verde — sem mover o cursor, sem instalar nada, sem admin.

⬇️ Baixar no GitHub 📦 Última release

✨ Recursos

🟢

Status sempre verde

Reseta o idle timer real do Windows — Teams, Slack e cia. continuam disponíveis.

🖱️

Não move o cursor

Pulso relativo (0,0): registra atividade sem mexer no seu mouse.

💤

Tela não apaga

Impede sleep e desligamento do monitor via API oficial de energia.

🕘

Janela de horário

Pulsa só nos dias e horas que você definir. Padrão: Seg–Sex, 9h–18h.

📌

Ícone na bandeja

Status colorido e ligar/desligar em um clique. Fecha pra bandeja.

🩺

Auto-recuperação

Watchdog relança o app se ele cair — sem precisar de administrador.

powershell -NoProfile -File .\jiggle-gui.ps1

🔬 Como funciona

A presença e o bloqueio de tela são governados pelo idle timer do Windows. O detalhe importante:

SetCursorPos(x, y)                → move o cursor, mas NÃO reseta o idle  ❌
mouse_event(MOUSEEVENTF_MOVE,0,0) → registra input, reseta o idle      ✅

O Jungle usa mouse_event com deslocamento (0,0): o Windows conta como atividade (status verde, tela acesa) sem mover o cursor. Em paralelo, SetThreadExecutionState impede sleep.

🛡️ Antivírus & uso consciente

Ferramentas que sintetizam input acionam heurísticas de antivírus/EDR — é esperado. O Jungle é transparente (um .ps1 auditável, sem lançadores ocultos), mas pode ser sinalizado em ambientes gerenciados. Não tente burlar o antivírus corporativo — fale com seu time de TI. E usar para simular presença quando você não está de fato trabalhando pode violar a política da empresa. Use por sua conta e risco.
